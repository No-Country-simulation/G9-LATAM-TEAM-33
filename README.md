## Descripción del proyecto

Crear una solución inteligente capaz de analizar patrones de consumo de energía eléctrica y generar información que ayude en la toma de decisiones relacionadas con la eficiencia energética.

La solución debe recibir información relacionada con el consumo de energía de una vivienda o pequeño establecimiento, como consumo mensual en kWh, horarios de mayor utilización, cantidad de equipos, perfil de uso de los ambientes y otros indicadores que el equipo considere relevantes.
